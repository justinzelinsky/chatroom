# Chatroom

A simple chatroom built with React, Redux, and Socket.IO featuring user authentication (WIP)

## Setup & Running

### Install

`npm install`

### Run UI
`npm run dev` 

### Run Server

`npm run server`

and navigate to `localhost:8082`
